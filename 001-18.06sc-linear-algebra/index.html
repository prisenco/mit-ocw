1. The Geometry of Linear Equations↑

This is easy.

Starting with three equations.

<#math-begin>2x - y = 0<#math-end>
<#math-begin>-x + 2y - z = -1<#math-end>
<#math-begin>-3y + 4z = 4<#math-end>

We need to normalize (is that the word?) these formulas so they can easily be mapped to a matrix.

<#math-begin>2x - y + 0z = 0<#math-end>
<#math-begin>-x + 2y - z = -1<#math-end>
<#math-begin>0z - 3y + 4z = 4<#math-end>

We can now easily put this in matrix form

<#math-begin> A=[[2,-1,0],[-1,2,-3],[0,-1, 4]] <#math-end> <#math-begin> B=[[0],[-1],[4]] <#math-end>
